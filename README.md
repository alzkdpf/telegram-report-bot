python telegram bot 연동 샘플 

#### setup

* insatll default virtual env

```type:shell

$virtualenv -p python3 venv
$. venv/bin/activate
```

* install module

```type:shell
$cd telbot-1.0.0
$pip install -r ./install_required.txt
```

* edit configuration

set your token value in '/config/service.cfg'

#### run
```type:shell
$cd telbot-1.0.0
$python ./bootstrap.py
```
